# mettarol-word
								English
Mettarol Word

Mettarol Word is a complete replacement for the popular Microsoft Word office application. It is designed specifically for users who prefer easy installation and no additional dependencies. Mettarol Word supports virtually any device running the Microsoft Windows operating system, from XP/Vista to the latest versions of Windows 10/11.

Key Features
File Format Compatibility
Mettarol allows you to create, edit, and view documents in common office formats, including:

.doc (Microsoft Word document format)
.docx
.txt
.odt (OpenDocument Text)
.html
Furthermore, the program allows you to save and export documents in the above formats, ensuring compatibility with most modern office applications.

System Requirements
The application is designed to be as accessible as possible to users of all skill levels. The only mandatory requirement is a Python interpreter installed on the computer. No other libraries or packages need to be installed!

Minimum hardware requirements:

Operating system: Windows XP SP3 / Vista / 7 / 8.x / 10 / 11
Processor: Intel Pentium IV with a frequency of at least 1 GHz
RAM: Minimum 512 MB RAM
Free disk space: Approximately 10 MB
Features
Completely free to use with no functionality limitations.
Quickly customize the interface appearance (color scheme, fonts).
Intuitive interface similar to MS Office.
Easy installation: running a single Python command installs the program.
Integration of standard text, table, and image formatting tools.
Mettarol Word was tested on two different OS versions: Windows Vista 32-bit and Windows 10 Pro 64-bit. The program ran successfully on both configurations, demonstrating consistent performance and functionality.

Mettarol Word

Программа Mettarol Word представляет собой полноценную замену популярному офисному приложению Microsoft Word. Она разработана специально для пользователей, предпочитающих простоту установки и отсутствие дополнительных зависимостей. Mettarol Word поддерживает работу практически на любых устройствах с операционной системой Microsoft Windows, начиная от версии XP/Vista и заканчивая последними версиями Windows 10/11.

Основные возможности
Совместимость форматов файлов
Меттарол позволяет создавать, редактировать и просматривать документы в распространенных офисных форматах, включая:

.doc (формат документов Microsoft Word)
.docx
.txt
.odt (OpenDocument Text)
.html
Кроме того, программа обеспечивает сохранение и экспорт документов в вышеперечисленные форматы, обеспечивая совместимость с большинством современных офисных приложений.

Требования к системе
Приложение разработано таким образом, чтобы быть максимально доступным пользователям любого уровня подготовки. Единственным обязательным требованием является наличие установленного интерпретатора Python на компьютере. Никаких других библиотек или пакетов устанавливать не потребуется!

Минимальные требования к аппаратному обеспечению:

Особенности
Полностью бесплатное использование без ограничений функциональности.
Возможность быстрой настройки внешнего вида интерфейса (цветовая схема, шрифты).
Интуитивно понятный интерфейс, аналогичный MS Office.
